# Multi-Agent Research Assistant Implementation Plan

We will build an autonomous research assistant using LangGraph, Google Gemini 2.5 Flash, Tavily Search, and Streamlit. The agent ecosystem will take a topic, search the web, analyze the findings, wait for your approval, write a report, and critique/revise it before outputting a formatted PDF.

### Core Architecture
#### [NEW] [agent/state.py](file:///c:/2026%20AI%20Projects/multi_agent_researcher/agent/state.py)
Defines the `AgentState` TypedDict to hold `topic`, `search_results`, `analysis`, `draft_report`, `critic_feedback`, and `revision_count`.

#### [NEW] [agent/nodes.py](file:///c:/2026%20AI%20Projects/multi_agent_researcher/agent/nodes.py)
Implements the 4 distinct agents:
1. `research_agent`: Uses Tavily API to fetch raw data.
2. `analysis_agent`: Uses Gemini to synthesize findings into an outline.
3. `writer_agent`: Uses Gemini to draft a full report based on the analysis.
4. `critic_agent`: Uses Gemini to review the report against quality heuristics and routes it back for revision if necessary.

#### [NEW] [agent/graph.py](file:///c:/2026%20AI%20Projects/multi_agent_researcher/agent/graph.py)
Wires the LangGraph node setup and defines the routing edges.
- Flow: `research_agent` -> `analysis_agent` -> `human_approval_pause` -> `writer_agent` <-> `critic_agent` -> END
- The `human_approval_pause` is achieved via LangGraph's `interrupt_before=["writer_agent"]` feature combined with `MemorySaver()`.

#### [NEW] [utils/pdf_export.py](file:///c:/2026%20AI%20Projects/multi_agent_researcher/utils/pdf_export.py)
Converts the final markdown generated by the `writer_agent` into a formatted PDF file using `fpdf2`. It will cleanly inject the Topic name, current Date, and Citations pulled from Tavily.

### Frontend
#### [NEW] [app.py](file:///c:/2026%20AI%20Projects/multi_agent_researcher/app.py)
The robust Streamlit user interface.
- Provides input for the Research Topic.
- Displays the agent execution in real-time (streaming LangGraph state).
- Provides an **"Approve & Write"** button when paused after analysis.
- Provides a **Download Report (PDF)** button for the final output.
